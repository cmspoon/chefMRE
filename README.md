# 🍴 chefMRE
    An algorithm determining the best MREs for creating secret recipes

---

## Database

The database includes all of MRE XLV (45) Menus:
- Entree
- Sides
- Desserts
- Beverages

It also includes a library of secret menu recipes:
- Ranger Pudding
- Birthday Cake
- Sour Patch Kids
